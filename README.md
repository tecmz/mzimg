# 魔众图床系统

魔众图床系统可以快速帮助可以搭建一个私有化图床系统，支持阿里云、腾讯云等各大云存储平台

## 运行环境


```
操作系统
Linux/Unix 或&nbsp;&nbsp;Windows
软件环境
Laravel 5.1的运行环境
Apache/Nginx , PHP 5.5.9+ / PHP 7.0 , MySQL 5.0+
```
